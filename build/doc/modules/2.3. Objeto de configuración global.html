2.3. Objeto de configuración global

Definido en: sitna.js:2799

Un objeto SITNA.Cfg está accesible para todas las instancias del la clase SITNA.Map.

Por tanto, se puede configurar un mapa asignando valores a las propiedades de ese objeto:

Ejemplo:

SITNA.Cfg.crs = "EPSG:4326";
SITNA.Cfg.initialExtent = [
  -2.84820556640625,
  41.78912492257675,
  -0.32135009765625,
  43.55789822064767
];
var map = new SITNA.Map("mapa");
